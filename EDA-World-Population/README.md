# Exploratory Data Analysis: World Population Dataset

This project performs an Exploratory Data Analysis(EDA) on the world population dataset using python and pandas.

**Key Highlights:**
- Data loaded and explored using pandas
- Numeric correlation matrix calculated for Population, Area and Growth Rate
- Grouped analysis by Continent
- Visualizations created including line graphs, boxplot and heatmap
- Insights derived for population trends and continent-wise comparisons

**Tools Used:**
- Python
- Pandas
- Matplotlib
- Seaborn
